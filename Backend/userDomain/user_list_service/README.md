### user-list-service

**Language:** Go  
**Framework:** Gin  
**ORM:** GORM  
**Database:** MySQL  
**Architecture:** RESTful API

## 📝 Description

This microservice exposes an API to **list all registered users**.  
Ideal for displaying users on the frontend or exporting user data.

## 🚀 Endpoint

- `GET /users/`  
  Returns a list of all available users.

## 🛠️ Technologies

- Go 1.21+
- Gin Gonic
- GORM
- MySQL
